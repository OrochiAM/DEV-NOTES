REQUEST RESPONSE CYCLE
1. Pocinje sa klijentom koji salje request koji sadrzi: metodu(GET), url(/api), podatke(path params)
2. Server handleuje request: filteruje podatke, baca greske, generise response
3. Response se salje klijentu i sadrzi: resurs, content type, status code, status message

Request Object (req u kodu) daje pristup requestu i sadrzi:
1. url
2. headere
3. podatke
4. metodu

Node je event driven

Event emitter

// import EventEmitter
import { EventEmitter } from 'node:events'

const customerDetails = {
  fullName: 'Meryl Sheep',
  email: 'baah@thedevilwearswool.com',
  phone: 12345678910
}

// create the emitter
const emailRequestEmitter = new EventEmitter()

// define the listener function
function generateEmail(customer) {
  console.log(`Email generated for ${customer.email}`)
}

// register the listener
emailRequestEmitter.on('emailRequest', generateEmail)
emailRequestEmitter.on('emailRequest', () => console.log('task assigned'))
emailRequestEmitter.on('emailRequest', () => console.log('email logged'))

// emit the event
setTimeout(()=> {
  emailRequestEmitter.emit('emailRequest', customerDetails)
}, 2000)
