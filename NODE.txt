HTTP Modul

import http from 'node:http' - ovako sa node: pa ime
import bez vijudavih zastava kada ne importujemo metode

const PORT = 8000 - broj porta

const server = http.createServer((req, res) => {
  res.write('This is some more data \n') - samo ispisuje ne zavrsava
  res.end('This is from the server', 'utf8', ()=> console.log('response end')) - slanje i zavrsavanje responsea
})

server.listen(PORT, () => console.log(`server running on port: ${PORT}`))

REQUEST RESPONSE CYCLE
1. Pocinje sa klijentom koji salje request koji sadrzi: metodu(GET), url(/api), podatke(path params)
2. Server handleuje request: filteruje podatke, baca greske, generise response
3. Response se salje klijentu i sadrzi: resurs, content type, status code, status message

Request Object (req u kodu) daje pristup requestu i sadrzi:
1. url
2. headere
3. podatke
4. metodu
